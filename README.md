World Happiness Report - Data Science Project
Jack Donahue - COMP3125 - Data Science Fundamentals

The goal of this project is to answer four key research questions using python based data analysis:

A) what impact does GDP have on happiness levels?
B) Do a country's main industries impact happiness?
C) What is the impact of the government system of a nation on happiness?
D) What is the impact of perceived government corruption on happiness?
